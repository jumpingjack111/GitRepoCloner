Build the APK yourself:

Follow these steps to build the APK from source:

  1. Clone the repository
  2. Run npm install
  3. Run npm run build
  4. Run npx cap add android
  5. Run npx cap sync android
  6. Run npx cap open android
  7. Build the APK in Android Studio

Future Installation Instructions

Once the APK is available, you'll be able to install it following these steps:

  1. Download the APK file when it becomes available
  2. Open your device's file manager and locate the downloaded APK
  3. Tap on the APK file to begin installation
  4. If prompted about security settings, go to Settings → Security → Enable "Unknown Sources"
  5. Return to the APK and continue installation
  6. Once installed, open the app and start studying!

Alternative: Install as PWA

In the meantime, you can install the web app as a Progressive Web App (PWA) on your device:

  1. Open the website in Chrome on your Android device
  2. Tap the menu button (three dots) in the top right
  3. Select "Add to Home Screen"
  4. Follow the prompts to install the PWA
